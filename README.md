# SKINHEAD — сайт бренда кожаных изделий

Адаптивный многостраничный сайт для демонстрации бренда натуральной кожи ручной работы. Реализован по макету Branding Agency, включает каталог с фильтрацией, корзину, профиль пользователя, интерактивный чат, карусели товаров и отзывов, а также динамическую подгрузку общих компонентов.  

Просмотр доступен по ссылке: [https://gina-eifman.github.io/skinhead/](https://gina-eifman.github.io/skinhead/)

## Особенности

- **Многостраничная структура**: главная (`index.html`), каталог (`catalog.html`), корзина (`cart.html`), профиль (`profile.html`), страница «О нас» (`about.html`);
- **Адаптивная вёрстка** под десктоп, планшет и мобильные устройства;
- **Компонентный подход**: шапка, подвал, чат и блок отзывов загружаются через `fetch` из отдельных HTML-файлов (`header.html`, `footer.html`, `chat.html`, `reviews.html`);
- **Каталог** с фильтрацией по категориям (обувь, сумки, аксессуары, куртки) и по типу (новинки, распродажа);
- **Поиск** по названию товара (перенаправляет на страницу каталога с результатами);
- **Корзина** на `localStorage`: добавление, удаление, изменение количества товаров, подсчёт итоговой суммы;
- **Профиль пользователя** (статические данные для просмотра);
- **Карусели**:
  - на главной — карусель товаров с названием, ценой и кнопкой «Сделать заказ»;
  - отзывы — пагинация по 3/2/1 отзыву в зависимости от ширины экрана, циклическое переключение;
- **Чат-виджет**: плавающая иконка, окно с демо-сообщениями, возможность отправки своих сообщений (бот отвечает через 1 секунду);
- **БЭМ-именование** классов;
- Все нерабочие ссылки/кнопки при клике показывают alert: «Это учебный проект. Функционал в разработке».

## Технологии

- HTML5
- CSS3 (адаптивная сетка, Flexbox, Grid, `backdrop-filter`)
- JavaScript (чистый, без библиотек)
- Fetch API (для подгрузки компонентов)
- Font Awesome 6 (иконки)

## Запуск

**Локально**:
1. Скачайте или склонируйте репозиторий.
2. Замените пути с абсолютных на относительные.
3. Откройте файл `index.html` в любом современном браузере.

**Онлайн**:  
Сайт доступен по ссылке [https://gina-eifman.github.io/skinhead/](https://gina-eifman.github.io/skinhead/)

## Примечания

- Проект **учебный**, не является реальным интернет-магазином. Все данные (цены, тексты, отзывы) демонстрационные.
- Для работы с корзиной используется `localStorage` — данные сохраняются после перезагрузки страницы.

## Лицензия

**Запрещено** использовать проект в любых целях. Сайт и репозиторий доступны для просмотра, **копирование** не допускается.
