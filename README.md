# SKINHEAD — сайт бренда кожаных изделий

Адаптивный двухстраничный сайт для демонстрации бренда натуральной кожи ручной работы. Реализован по макету Branding Agency (главная страница + страница «О нас»), включает интерактивный чат, карусели товаров и отзывов, а также динамическую подгрузку общих компонентов. Просмотр доступен по ссылке [https://gina-eifman.github.io/skinhead/](https://gina-eifman.github.io/skinhead/).

## Особенности
- **две страницы**: главная (`index.html`) и «О нас» (`about.html`);
- **адаптивная вёрстка** под десктоп, планшет и мобильные устройства;
- **компонентный подход**: шапка, подвал, чат и блок отзывов загружаются через `fetch` из отдельных HTML-файлов (`header.html`, `footer.html`, `chat.html`, `reviews.html`). Это облегчает поддержку и переиспользование;
- **карусели**:
  - на главной — карусель товаров с названием и ценой поверх изображения, точками навигации и кнопкой «Сделать заказ»;
  - отзывы — пагинация по 3/2/1 отзыва в зависимости от ширины экрана, циклическое переключение (после последней страницы идёт первая);
- **чат-виджет**: плавающая иконка, при клике открывается окно с демо-сообщениями. Пользователь может отправлять сообщения — бот отвечает через 1 секунду;
- **БЭМ-именование** классов для чистоты и предсказуемости стилей;
- **все нерабочие ссылки/кнопки** (каталог, добавление в корзину, соцсети и т.п.) при клике показывают alert: *«Это учебный проект. Функционал в разработке.»*.

## Технологии
- HTML5;
- CSS3 (адаптивная сетка, Flexbox);
- JavaScript (чистый, без библиотек);
- Fetch API (для подгрузки компонентов);
- Font Awesome 6 (иконки);

## Структура проекта
- index.html # Главная страница
- about.html # Страница «О нас»
- styles/
  - common.css # Общие стили (шапка, подвал, чат, отзывы)
  - index.css # Стили только для главной страницы
  - about.css # Стили только для страницы «О нас»
- scripts/
  - data.js # Общие данные (товары, отзывы, функция eduAlert)
  - common.js # Загрузка и инициализация общих компонентов
  - index.js # Логика главной страницы (товары, карусель)
  - about.js # Логика страницы «О нас» (контактные кнопки)
- components/
  - header.html # Разметка шапки
  - footer.html # Разметка подвала
  - chat.html # Разметка чата
  - reviews.html # Разметка блока отзывов
- assets/ # Папка с изображениями

## Запуск
Локально:
1. Скачайте или склонируйте репозиторий.
2. Откройте файл `index.html` в любом современном браузере.
Онлайн:
Сайт располагается по ссылке [https://gina-eifman.github.io/skinhead/](https://gina-eifman.github.io/skinhead/)
**Важно**: Замените абсолютные пути к файлам на относительные.

## Примечания
Проект **учебный**, не является реальным интернет-магазином. Все данные (цены, тексты, отзывы) демонстрационные.
